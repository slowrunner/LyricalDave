*** Install Python API for Ollama
```
sudo apt install python3-ollama
# OLD: pip install ollama --break-system-packages
```

*** Remove a model ***
```
ollama list
ollama rm model_name  eg.: qwen2.5:0.5b
```

*** See model(s) ollama is running ***
```
ollama ps
```

=== Remove ollama ===  
```
sudo systemctl stop ollama
sudo systemctl disable ollama
sudo rm /etc/systemd/system/ollama.service
sudo rm $(which ollama)
sudo rm -rf /usr/share/ollama
sudo rm -rf /etc/ollama
sudo rm -rf /usr/local/lib/ollama  (3G)
sudo rm -rf /home/.ollama*

sudo userdel ollama
sudo groupdel ollama
```
